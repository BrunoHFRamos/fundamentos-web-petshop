# Jaguara's Petshop - Plataforma Web (Fase 1)

Este projeto foi desenvolvido como requisito para a disciplina **Fundamentos do Desenvolvimento Web da Faculdade PUCRS**. 
A Fase 1 foca exclusivamente na construção da estrutura semântica e do conteúdo principal do petshop, utilizando apenas HTML Puro.

--------------------------------------------------------------------------------------------------------------------------------------------------------------

# Funcionalidades

# 1. Catálogo de Conteúdo
O sistema apresenta a visualização do catálogo de produtos e serviços, conforme os requisitos da disciplina:

**Produtos:** Seis produtos listados, organizados e agrupados por categorias: **Acessórios**, **Rações não perecíveis** e **Produtos de higiene e limpeza**.
**Detalhes:** Cada item exibe a **foto**, a **descrição** e o **valor**.
**Serviços:** Detalhamento do Banho e Tosa, especificando a descrição, valor e a disponibilidade de tele-busca.

## 2. Estrutura e Restrições Técnicas
O projeto é baseado em um único arquivo ("index.html").

**Tecnologia:** Desenvolvido exclusivamente com **HTML5**.
**Restrições:** **Não há uso de CSS, Bootstrap ou qualquer outra biblioteca de customização/estilo.** O design apresentado é o padrão do navegador.
**Interatividade:** **Não há implementação de JavaScript**.
